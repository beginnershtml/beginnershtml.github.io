TABLES in HTML

The HTML tables are created using the <table> tag in which the <tr> tag is used to create table rows and <td> tag is used to create data cells. The elements under <td> are regular and left aligned by default

  • The <table> tag is used to create a table.
  • In HTML, a table is considered as a group of rows containing each group of cells.
  • There can be many columns in a row.
  • HTML tables should be used for tabular data only, but they are also used for creating layout web pages.
  • Table tag : <TABLE> </TABLE&g
  • The content which we write between these tags will be displayed within the table.

Below are some of the tags used in the table:

  1. <tr> TAG:

    2. The tag is used to define a row in the HTML table. we will start the row with the beginning by row tag and then build the row by creating each cell, and when we finish all the cells for a row, we will then close the row with the ending row tag </tr>.

  2. <td> TAG:

    3. The tag is used to define the data for the cell in the HTML table. We will create each cell with the beginning cell tag <td> and then add the content or data to the cell and then close the cell with the ending cell tag </td>.

  3. <th> TAG:

    4. The tag is used to define the header cell in an HTML table. The header cell in the table is used to provide information for the remaining cells of the column.
    Usually, it is placed at the top of the cells.
