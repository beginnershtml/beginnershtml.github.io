CHANGE THE LINK'S COLOUR

HOW to change link colour

To change the color of links in HTML, use the CSS property color. 
Use it with the style attribute. The style attribute specifies an inline style for an
element.Use the style attribute with the CSS property color to change the link color.
Just keep in mind, the usage of style attribute overrides any style set globally. 
It will override any style set in the HTML. tag or external style sheet.
