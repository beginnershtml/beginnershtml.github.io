External -

by using a element to link to an external CSS file

An external style sheet is used to define the style for many HTML pages.

To use an external style sheet, add a link to it in the section of each HTML page:

The rel attribute

The first of the two indispensable attributes is the rel attribute.
You will use this attribute to tell the browser what the relationship is with the imported file.

You’ll write rel=”stylesheet” to tell the browser that you are importing a stylesheet.
